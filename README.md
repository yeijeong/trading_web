## 환경세팅
- Java 11 version
- Java Script
- Spring Boot
- Maria DB
- BootStrap

## 아쉬운점
- 시간이 많지 않아 제대로 된 실시간 그래프를 구현하지 못했다. 레디스를 사용하여 만들고 싶었으나 시간 부족으로 인하여 구현하지 못한 부분이 아쉽다.
- 그래프를 불러오는 속도가 조금 느린 부분이 아쉽다.
